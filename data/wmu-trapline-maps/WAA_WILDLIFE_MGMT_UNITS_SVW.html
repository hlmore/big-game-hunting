Wildlife Management Units

5206 views (160 recent)

Published by the Ministry of Forests, Lands, Natural Resource Operations and Rural Development - Wildlife and Habitat
Licensed under Open Government Licence - British Columbia

The Province is divided into nine administrative regions, having a total of 225 wildlife management units (WMU) for the purpose of efficient game management.

View in iMapBC

Additional Information

Purpose

This dataset was created to provide information to support informed management decisions for land and natural resource planning including wildlife management, protection of biodiversity and/or BC's resource based industries.Management Units are used to set regulations for hunting and fishing.
Lineage Statement

Data was provisional in the fall of 1994.

Object Description

Object Name: WHSE_WILDLIFE_MANAGEMENT.WAA_WILDLIFE_MGMT_UNITS_SVW

Short Name: WAAWMU_SVW

Column Name Short Name Data Type Data Precision Comments
WILDLIFE_MGMT_UNIT_ID WMU_ID VARCHAR2 4 The Primary Uniqe Identifier for the management unit. Consists of the Environment Region number, followed by a dash followed by a two-digit number unique within the region, e.g., 4-2.
FEATURE_CODE FEAT_CODE VARCHAR2 10 Contains an alphanumeric value based on the Canadian Council of Surveys and Mapping’s (CCSM) system for classification of geographic features, e.g., WI02800110.
REGION_RESPONSIBLE REG_RESPON NUMBER 16 The region code responsible for management on the wildlife unit, e.g., 4.
REGION_RESPONSIBLE_NAME REG_R_NAME VARCHAR2 100 The region name responsible for management on the wildlife unit, e.g., Kootenay.
GAME_MANAGEMENT_ZONE_ID GMZ_ID VARCHAR2 5 The primary unique numeric identifier for the zone. Starts with a WLAP region number, e.g., 4Ea.
GAME_MANAGEMENT_ZONE_NAME GMZ_NAME VARCHAR2 50 The primary unique name identifier for the zone. Starts with a WLAP region number, e.g., Fernie.
GEOMETRY GEOMETRY SDO_GEOMETRY 64 The column used to reference the spatial coordinates defining the feature.
FEATURE_AREA_SQM AREA_SQM NUMBER 19 The system calculated area of a two-dimensional polygon in square meters.
FEATURE_LENGTH_M FEAT_LEN NUMBER 19 The system calculated length or perimeter of a geometry in meters.
OBJECTID OBJECTID NUMBER 38 A column required by spatial layers that interact with ESRI ArcSDE. It is populated with unique values automatically by SDE.
SE_ANNO_CAD_DATA BLOB 4000 A binary column used by spatial tools to store annotation, curve features and CAD data when using the SDO_GEOMETRY storage data type.
